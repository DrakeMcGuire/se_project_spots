# Project: Spots

### Description 

Spots is an interactive image sharing site that is optimized for large desktop and mobile phone screens. It also features buttons for editing profiles, creating posts, and the ability to like photos.
  
**Technologies**
  
* HTML
* CSS
* Responsive Design
